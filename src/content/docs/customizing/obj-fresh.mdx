When you want to figure out which object are recently created the fresh ones
are colored green. It only works with `get` subcommand.

You can customize threshold level by using environment variable `KUBECOLOR_OBJ_FRESH`
or `~/.kube/color.yaml` config field `objFreshThreshold`
with value format [time.Duration](https://golang.org/pkg/time/#ParseDuration).

<Tabs syncKey="config">
  <TabItem label="Environment variables">
  ```bash
  export KUBECOLOR_OBJ_FRESH="10h"
  ```
  </TabItem>
  <TabItem label="Config file">
  ```yaml
  # ~/.kube/color.yaml
  objFreshThreshold: 10h
  ```
  </TabItem>
</Tabs>
